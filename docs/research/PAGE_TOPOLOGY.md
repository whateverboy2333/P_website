# Page Topology

1. Wallpaper layer: black full-page tiled pattern with colored circular `05/2` motif.
2. CRT overlay: fixed topmost scan-line layer, user-toggleable.
3. Main OS window: centered, classic beveled chrome, gradient titlebar.
4. Desktop content area: white lined inset with app icons arranged in a grid and trash icon anchored bottom-right.
5. Footer command strip: two wide beveled buttons, `About this website` and `Last updated May 8`.
6. Statusbar: copyright left, CRT/LCD switch right.
7. Popup layer: absolute overlay for opened app windows, draggable and closable.
