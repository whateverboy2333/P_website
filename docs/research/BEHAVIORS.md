# Behaviors

- Main page is a centered retro desktop window over a tiled black background.
- Top-left titlebar dot toggles the main content body.
- Middle titlebar button changes wallpaper; right titlebar button powers off into a black screen state.
- Desktop icons open popup windows, while Blog and the footer update link navigate externally.
- Popup windows can be focused, closed, and dragged by their titlebars.
- Cestino opens a small dialog and cycles nostalgic messages on repeated activation.
- CRT/LCD switch toggles fixed scan-line overlay.
- Responsive behavior: desktop uses a 6-column icon grid in a 640px window; mobile switches to a 3-column grid and stacked footer buttons.
